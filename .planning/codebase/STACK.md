# Tech Stack & Tools

## Frontend

- **Framework**: React 19 (via Vite)
- **Routing**: React Router v7
- **Styling**: Tailwind CSS v4, Lucide React (Icons)
- **State/Caching**: Context API, Workbox (PWA), local storage.
- **Offline DB**: RxDB / Dexie for local-first/offline tracking.
- **Animations**: Framer Motion.
- **Other**: React Loading Skeleton, React Markdown.

## Backend

- **Framework**: Express.js
- **Runtime**: Node.js 18+ (uses native `fetch`)
- **Language**: TypeScript (ts-node / tsc)
- **Database**: MongoDB (via Mongoose ORM)
- **Auth**: JWT (JSON Web Tokens), Google OAuth2
- **Testing**: Vitest, mongodb-memory-server (integration tests).
