## 🚀 Quick Start

### Prerequisites
- **Node.js** (v14 or higher)

### Setup in 3 Steps

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Configure your API key**
   - Open `.env.local`
   - Add your Gemini API key:
     ```
     GEMINI_API_KEY=your_key_here
     ```

3. **Run the app**
   ```bash
   npm run dev
   ```

Your app will be live at `http://localhost:3000`
